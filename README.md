# Time-Rhythm
《时律》是一款面向时间盲视、拖延倾向、执行功能障碍和高压多任务人群的 AI 视觉日程管理应用，它用 24 小时可视化时间圆环替代传统清单，让用户直观看见时间流逝与任务占比；同时通过 AI 自动拆解复杂任务、生成父子任务结构、插入休息节奏、检测时间冲突、在任务超时时进行温和提醒与动态顺延，帮助用户把“我知道要做，但不知道怎么开始”转化为“下一步该做什么、现在还剩多少时间、是否需要休息”的清晰行动路径。这个产品要解决的核心痛点，是用户面对大任务时容易启动困难、对时间没有真实感、经常低估任务耗时、排程过满后迅速疲惫，以及现有效率工具过于表格化、理性化、缺少情绪安抚与视觉反馈的问题。
